2. 음성분야 AI

3.이미지 분야 AI
이미지 분야에서 요즘 가장큰 이슈는 바로 의료이다. 의료분야 AI는 X-ray, CT 촬영 이미지등을 파악하여 그전에 의사들이 직접보고 판단하던것을 이제는 인공지능으로 분석하여 더 빠르고 더 정확한 진찰을 가능하게 하고있다.

4 자율주행 분야 AI
